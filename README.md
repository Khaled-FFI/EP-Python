# EP-Python

### Objectif
Créer un script Python capable d'organiser automatiquement les fichiers d'un dossier selon leur extension.

### Technologies
- Python 3

### Fonctionnalités
- Détection automatique des extensions
- Création des dossiers si nécessaire
- Déplacement des fichiers dans les bons dossiers
- Support des images, PDF, textes, vidéos et archives

### Ce que j'ai appris
- Manipuler les fichiers avec Python
- Utiliser les modules `os` et `shutil`
- Automatiser une tâche simple mais utile
- Organiser un projet Python sur GitHub

### Utilisation
1. Télécharger le fichier `organizer.py`
2. Lancer le script :
   ```bash
   python organizer.py
